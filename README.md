# github-approve-shortcut

Tampermonkey script that binds keyboard shortcuts to approve PRs:

- `option+shift+u`: approve with a `utACK`
- `option+shift+t`: approve with a `tACK`

### Usage

Copy-paste JS in tampermonkey.js to a [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) User Script, enable and use on any Github PR page.
